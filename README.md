## Доступные игры

1. **brain-even** - Определить, является ли число чётным
2. **brain-calc** - Вычислить результат математического выражения
3. **brain-gcd** - Найти наибольший общий делитель двух чисел
4. **brain-progression** - Найти пропущенное число в арифметической прогрессии
5. **brain-prime** - Определить, является ли число простым


## Установка:

### Требования
- Node.js версии 18 или выше
- npm

### Установка пакета
```bash
# Клонируйте репозиторий
git clone https://github.com/Palaye-Royale/frontend-project-44.git

# Перейдите в папку проекта
cd frontend-project-44

# Установите зависимости
make install

# Установите пакет глобально
sudo npm link

### После установки доступны команды:

- brain-games    # Приветствие
- brain-even     # Запуск игры "Проверка на чётность"
- brain-calc     # Запуск игры "Калькулятор"
- brain-gcd      # Запуск игры "НОД"
- brain-progression # Запуск игры "Прогрессия"
- brain-prime    # Запуск игры "Простое ли число?"
